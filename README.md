# Floating Glass Navbar

A sleek **floating navigation bar** inspired by modern mobile UI design.

This component features a **glassmorphism style container**, smooth shadows, and a **glowing center action button** perfect for mobile-first web applications.

---

## ✨ Features

- Floating bottom navigation bar
- Glassmorphism UI style
- Center floating action button
- Smooth glow effect
- Minimal modern icons
- Mobile-first design
- Clean and lightweight implementation

---

## 📱 UI Components

This project includes:

- Bottom navigation container
- Active navigation indicator
- Center floating action button
- Icon navigation items
- Gradient hero section

---

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript
- Flexbox
- CSS blur & shadow effects

---

## 📱 Responsive Design

Designed primarily for:

- Mobile devices
- Mobile web apps
- Progressive Web Apps (PWA)
